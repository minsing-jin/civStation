# 설치

이 페이지는 예전 설치 페이지를 docs 포털 안에서 보존한 것입니다.

## stable release

```sh
uv add civStation
```

또는 `pip` 사용:

```sh
pip install civStation
```

## source에서 설치

소스는 [GitHub repo](https://github.com/minsing-jin/civStation)에서 받을 수 있습니다.

```sh
git clone https://github.com/minsing-jin/civStation.git
cd civStation
uv pip install .
```

현재 기준의 더 실전적인 안내는 [시작하기 / 설치](getting-started/installation.md)를 보세요.
