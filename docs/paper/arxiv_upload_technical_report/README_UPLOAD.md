# arXiv Upload Notes

This directory is the canonical source package for uploading the current CivStation technical report.

## Primary file

- `main.tex`

## Intended public framing

This document should be described as:

- a technical report
- an experimental systems note
- a design and lessons-learned artifact

It should **not** be described as:

- a validated benchmark paper
- a strong empirical systems result
- a completed long-horizon Civ6 evaluation study

## Recommended title

`CivStation: Technical Report on an Experimental Layered Stack for Civilization VI Computer Use`

## Recommended description

This report documents the architecture, instrumentation, and current limitations of CivStation as an experimental stack for long-horizon game computer use. It preserves design choices and engineering lessons, rather than claiming benchmark maturity.
