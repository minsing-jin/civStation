# 安装

这页是旧安装页面在 docs 门户中的保留版本。

## stable release

```sh
uv add civStation
```

或者使用 `pip`：

```sh
pip install civStation
```

## 从源码安装

源码可以从 [GitHub repo](https://github.com/minsing-jin/civStation) 获取。

```sh
git clone https://github.com/minsing-jin/civStation.git
cd civStation
uv pip install .
```

如果你需要更实用、更新的说明，请看 [Getting Started / Installation](getting-started/installation.md)。
